Let's Learn JavaScript's ReGex Quantifiers

The code in the external JS file is as follows.

The n+ quantifier matches any string that contains at least one n.


      let text = "Hellooo World! Hello W3Schools!";
      let pattern = /o+/g;
      let result = text.match(pattern);
      
      document.getElementById("demo1").innerHTML = result;

The n* quantifier matches any string that contains zero or more occurrences of n.


      text = "Hellooo World! Hello W3Schools!";
      pattern = /lo*/g;
      result = text.match(pattern);

      document.getElementById("demo2").innerHTML = result;

The n? quantifier matches any string that contains zero or one occurrences of n.


      text = "1, 100 or 1000?";
      pattern = /10?/g;
      result = text.match(pattern);
      
      document.getElementById("demo3").innerHTML = result;

The n{X} quantifier matches any string that contains a sequence of X n's. X must be a number. 


      text = "100, 1000 or 10000?";
      pattern = /\d{4}/g;
      result = text.match(pattern);
      
      document.getElementById("demo4").innerHTML = result;

The n{X,Y} quantifier matches any string that contains a sequence of X to Y n's. X and Y must be a number. 


      text = "100, 1000 or 10000?";
      pattern = /\d{3,4}/g;
      result = text.match(pattern);
      
      document.getElementById("demo5").innerHTML = result;

The n{X,} quantifier matches any string that contains a sequence of at least X n's. X must be a number. 


      text = "100, 1000 or 10000?";
      pattern = /\d{3,}/g;
      result = text.match(pattern);
      
      document.getElementById("demo6").innerHTML = result;

The ^n quantifier matches any string with n at the beginning of it.


      text = "Is this his";
      pattern = /^Is/g;
      result = text.match(pattern);
      
      document.getElementById("demo7").innerHTML = result;

The n$ quantifier matches any string with n at the end of it.


      text = "Is this his";
      pattern = /is$/;
      result = text.match(pattern);
      
      document.getElementById("demo8").innerHTML = result;

The ?=n quantifier matches any string that is followed by a specific string n.


      text = "Is thisall there is is";
      pattern = /is(?=all)/;
      result = text.match(pattern);
      
      document.getElementById("demo9").innerHTML = result;

The ?!n quantifier matches any string that is not followed by a specific string n.


      text = "Is this all there is is";
      pattern = /is(?! all)/gi;
      result = text.match(pattern);
      
      document.getElementById("demo10").innerHTML = result;
