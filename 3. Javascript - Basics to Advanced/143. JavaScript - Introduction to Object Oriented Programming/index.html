Let's Learn Object Oriented Programming in JavaScript

Open Console to see the output for the JavaScript code.

The code in the external JS file is as follows.


      console.clear()

      // Create an Object
      const obj = {
        firstName: "Abhishek",
        lastName: "Soren",

        display: () => {
          console.log(obj.firstName)
          console.log(obj.lastName)
        },

        displayAgain: function() {
          return `${this.firstName} ${this.lastName}`
        }
      }

      obj.display()

      const newObj = obj

      newObj.firstName = "Potato"

      newObj.display()
      obj.display()

      // Invoke displayAgain
      console.log(obj.displayAgain())
